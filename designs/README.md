# 企业知识库问答系统 - UI设计

本文件夹包含企业知识库问答系统的两个核心页面的UI设计原型。

## 文件结构

```
designs/
├── README.md                    # 设计说明文档
├── knowledge-management.html    # 文档管理页面
└── chat-agent.html              # AI智能助手聊天页面
```

## 页面说明

### 1. 文档管理页面 (knowledge-management.html)

**功能特性：**
- 🔍 **搜索功能** - 搜索文档
- ⬆️ **上传功能** - 上传文档
- 📋 **文档列表** - 表格形式展示文档信息

**精简内容：**
- ✗ 移除了左侧分类导航
- ✗ 移除了统计卡片
- ✗ 移除了视图切换和筛选标签
- ✗ 移除了多余的导航项（只保留文档管理和智能问答）
- ✗ 移除了用户头像
- ✓ 保留：搜索、上传、文档列表

### 2. AI智能助手聊天页面 (chat-agent.html)

**功能特性：**
- 💬 **实时聊天** - 与AI助手进行对话交流
- 📚 **引用来源** - AI回答中标注参考的文档来源

**精简内容：**
- ✗ 移除了左侧会话列表
- ✗ 移除了右侧参考文档边栏
- ✗ 移除了RAG开关
- ✗ 移除了思考过程展示
- ✗ 移除了快捷问题
- ✗ 移除了消息操作按钮
- ✗ 移除了多余的导航项
- ✓ 保留：核心聊天界面、消息气泡、引用来源

## 设计规范

### 色彩系统
- **主色调**：紫色渐变 `#667eea` → `#764ba2`
- **背景色**：浅灰 `#f5f7fa`、白色 `#ffffff`
- **文字色**：深灰 `#1a1a2e`、中灰 `#374151`、浅灰 `#6b7280`
- **状态色**：成功 `#059669`、警告 `#d97706`

### 导航
- 仅保留两个Tab：文档管理、智能问答

## 如何使用

1. **直接预览**：在浏览器中打开 HTML 文件即可查看设计效果
2. **开发参考**：可作为前端开发的视觉参考和原型

## 技术说明

- 纯 HTML + CSS 实现，无需任何框架依赖
- 使用 CSS Flexbox 进行布局
- 包含完整的状态样式（hover、active等）
