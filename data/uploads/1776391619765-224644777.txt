# API接口文档 - Version 2.0

## 1. 概述

本文档描述企业知识库系统的后端API接口，供前端开发和第三方集成使用。

### 1.1 基础信息

- **Base URL**: https://api.kb.company.com/v2
- **协议**: HTTPS
- **数据格式**: JSON
- **字符编码**: UTF-8

### 1.2 认证方式

**Bearer Token认证：**
```
Authorization: Bearer <access_token>
```

**获取Token：**
- 通过SSO登录获取
- Token有效期：24小时
- 过期后需刷新获取新Token

### 1.3 通用响应格式

**成功响应：**
```json
{
  "ok": true,
  "data": { ... },
  "request_id": "req_abc123xyz"
}
```

**错误响应：**
```json
{
  "ok": false,
  "error": "错误描述信息",
  "error_code": "INVALID_PARAM",
  "request_id": "req_abc123xyz"
}
```

### 1.4 错误码说明

| 错误码 | HTTP状态码 | 说明 |
|--------|-----------|------|
| SUCCESS | 200 | 请求成功 |
| INVALID_PARAM | 400 | 参数错误 |
| UNAUTHORIZED | 401 | 未授权/Token无效 |
| FORBIDDEN | 403 | 无权限访问 |
| NOT_FOUND | 404 | 资源不存在 |
| RATE_LIMITED | 429 | 请求频率超限 |
| INTERNAL_ERROR | 500 | 服务器内部错误 |

## 2. 会话管理API

### 2.1 创建会话

**接口地址**: `POST /sessions`

**请求参数：**
```json
{
  "user_id": "user_123",
  "title": "技术问题咨询",
  "metadata": {
    "source": "web",
    "device": "chrome"
  }
}
```

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "session_id": "session_456",
    "user_id": "user_123",
    "title": "技术问题咨询",
    "created_at": "2024-03-15T10:30:00Z"
  }
}
```

### 2.2 获取会话列表

**接口地址**: `GET /sessions`

**请求参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| user_id | string | 是 | 用户ID |
| limit | int | 否 | 返回数量，默认20 |
| offset | int | 否 | 偏移量，默认0 |

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "total": 100,
    "sessions": [
      {
        "session_id": "session_456",
        "title": "技术问题咨询",
        "last_message": "请问API怎么调用？",
        "updated_at": "2024-03-15T10:35:00Z"
      }
    ]
  }
}
```

### 2.3 获取会话详情

**接口地址**: `GET /sessions/{session_id}`

**路径参数：**
- `session_id`: 会话ID

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "session_id": "session_456",
    "user_id": "user_123",
    "title": "技术问题咨询",
    "messages": [
      {
        "role": "user",
        "content": "请问API怎么调用？",
        "created_at": "2024-03-15T10:30:00Z"
      },
      {
        "role": "assistant",
        "content": "您好，API调用方式如下...",
        "created_at": "2024-03-15T10:30:05Z"
      }
    ]
  }
}
```

### 2.4 删除会话

**接口地址**: `DELETE /sessions/{session_id}`

**路径参数：**
- `session_id`: 会话ID

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "session_id": "session_456",
    "deleted": true
  }
}
```

## 3. 聊天API

### 3.1 发送消息（SSE流式响应）

**接口地址**: `POST /chat`

**Query参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| message | string | 是 | 用户消息内容 |
| user_id | string | 是 | 用户ID |
| session_id | string | 是 | 会话ID |
| rag | string | 否 | 是否启用RAG，"1"启用，"0"禁用 |

**请求示例：**
```
POST /chat?message=你好&user_id=user_123&session_id=session_456&rag=1
```

**SSE响应格式：**
```
data: {"model": "deepseek-chat"}

data: {"text": "你"}

data: {"text": "好"}

data: {"text": "！"}
```

**SSE字段说明：**
- `model`: 模型名称（第一条消息）
- `text`: 文本片段（流式输出）
- `error`: 错误信息（出错时）

### 3.2 发送消息（非流式）

**接口地址**: `POST /chat/non-streaming`

**请求体：**
```json
{
  "message": "你好",
  "user_id": "user_123",
  "session_id": "session_456",
  "rag": true
}
```

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "content": "你好！有什么可以帮助你的？",
    "model": "deepseek-chat",
    "sources": [
      {
        "document_id": "doc_789",
        "title": "常见问题解答",
        "relevance": 0.95
      }
    ]
  }
}
```

## 4. 文档管理API

### 4.1 上传文档

**接口地址**: `POST /documents/upload`

**Content-Type**: `multipart/form-data`

**Form参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| file | File | 是 | 上传的文件 |
| user_id | string | 是 | 用户ID |
| title | string | 否 | 文档标题，默认使用文件名 |

**请求示例（curl）：**
```bash
curl -X POST https://api.kb.company.com/v2/documents/upload \
  -H "Authorization: Bearer <token>" \
  -F "file=@/path/to/document.pdf" \
  -F "user_id=user_123" \
  -F "title=技术文档"
```

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "document_id": "doc_789",
    "filename": "技术文档.pdf",
    "size": 1048576,
    "mime_type": "application/pdf",
    "title": "技术文档",
    "created_at": "2024-03-15T10:30:00Z"
  }
}
```

### 4.2 获取文档列表

**接口地址**: `GET /documents`

**请求参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| user_id | string | 是 | 用户ID |
| keyword | string | 否 | 搜索关键词 |
| limit | int | 否 | 返回数量，默认20 |
| offset | int | 否 | 偏移量，默认0 |

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "total": 100,
    "documents": [
      {
        "document_id": "doc_789",
        "title": "技术文档.pdf",
        "filename": "技术文档.pdf",
        "size": 1048576,
        "mime_type": "application/pdf",
        "indexed": true,
        "created_at": "2024-03-15T10:30:00Z"
      }
    ]
  }
}
```

### 4.3 获取文档详情

**接口地址**: `GET /documents/{document_id}`

**路径参数：**
- `document_id`: 文档ID

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "document_id": "doc_789",
    "title": "技术文档.pdf",
    "filename": "技术文档.pdf",
    "size": 1048576,
    "mime_type": "application/pdf",
    "user_id": "user_123",
    "indexed": true,
    "chunks_count": 50,
    "created_at": "2024-03-15T10:30:00Z"
  }
}
```

### 4.4 下载文档

**接口地址**: `GET /documents/{document_id}/download`

**路径参数：**
- `document_id`: 文档ID

**响应：**
- 成功：返回文件流，Content-Type为文件MIME类型
- 失败：返回错误JSON

### 4.5 删除文档

**接口地址**: `DELETE /documents/{document_id}`

**路径参数：**
- `document_id`: 文档ID

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "document_id": "doc_789",
    "deleted": true
  }
}
```

## 5. RAG检索API

### 5.1 纯文本导入

**接口地址**: `POST /rag/ingest`

**请求体：**
```json
{
  "user_id": "user_123",
  "title": "FAQ文档",
  "text": "这里是纯文本内容..."
}
```

**响应示例：**
```json
{
  "ok": true,
  "document_id": "doc_789",
  "chunks": 50
}
```

### 5.2 向量搜索

**接口地址**: `POST /rag/search`

**请求体：**
```json
{
  "user_id": "user_123",
  "query": "搜索关键词",
  "top_k": 5
}
```

**响应示例：**
```json
{
  "ok": true,
  "data": {
    "hits": [
      {
        "document_id": "doc_789",
        "chunk_id": "chunk_123",
        "content": "相关文本片段...",
        "score": 0.95
      }
    ]
  }
}
```

## 7. 附录

### 7.1 限流规则

| API | 限流 |
|-----|------|
| 聊天接口 | 60次/分钟/用户 |
| 文档上传 | 10次/分钟/用户 |
| 其他接口 | 120次/分钟/用户 |

### 7.2 文件格式支持

| 格式 | 扩展名 | MIME类型 |
|------|--------|----------|
| PDF | .pdf | application/pdf |
| Word | .doc, .docx | application/msword, application/vnd.openxmlformats... |
| PowerPoint | .ppt, .pptx | application/vnd.ms-powerpoint... |
| 纯文本 | .txt, .md | text/plain, text/markdown |
